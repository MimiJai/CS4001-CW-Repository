PROJECT TITLE: GadgetShop Project 
PURPOSE OF PROJECT: This project is a JavaFX-based application developed as part of the CS4001 programming class coursework. This coursework aims to demonstrate object-oriented programming concepts, including polymorphism, inheritance, and ArrayLists. The application allows users to create and manage different types of gadgets, specifically Mobile phones and MP3 players, through a GUI. 
DATE: April 2026
HOW TO START THIS PROJECT: To run the project, open it in a Java IDE such as BlueJ, compile the classes, and then run the GadgetShop to launch the graphical interface. The program can also be run using the command prompt by compiling and running the files with the JavaFX module path. 
AUTHORS: Jaimie Mounsmie 
USER INSTRUCTIONS: Enter the required details into the input fields, then select the appropriate button to perform the desired action. Use “Add Mobile” to create a mobile device and “Add MP3” to create an MP3 player. The “Display All” button shows all stored gadgets in the system. To interact with a specific gadget, enter its display number. The “Make A Call” function works with mobile devices, while “Download Music” works with MP3 devices. If you enter an invalid display number, the program will show an error message to guide you.
